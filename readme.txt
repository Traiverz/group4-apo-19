*копирование репозитория
-переходим в папку где создадим копию проекта
вводим git clone [ссылка]

*переходим в папку с проектом
cd [название проекта]

*статус нашего проекта
git status

*добавление файла на репозиторий
git add [название файла]
git commit -m "описание изменений"
git push


*скачивание файла другому пользователю
gip pull



*закинуть 2 файла:
git add -A

проверка
git status
